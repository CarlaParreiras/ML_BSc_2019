Instruções:

MyML: Módulo (Julia) de funções auxiliares necessárias aos modelos, como geração de dados aleatórios e cluster.
KNN: Código (Julia) que usa o algoritmo KNN para regressão e salva o plot em um pdf caso as dimensões dos dados sejam menores que 3.
k-RBF-Network: Código (Julia) de implementação da RBF Network com escolha de r por cross-validation e plot da regressão feita.
